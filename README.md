
  Diário de Estudos Interativo 📚

Objetivo:
Permitir que o estudante planeje tarefas, marque como concluídas e acompanhe seu progresso, tudo de forma visual e gamificada.

Página Inicial:

* Cabeçalho com título do diário
* Campo de entrada para digitar a tarefa
* Seleção de prioridade (alta, média, baixa)
* Botão para adicionar tarefa
* Lista das tarefas adicionadas
* Design limpo, mobile-first, cores que indicam prioridade

Funcionalidades JS:

* Adicionar tarefa à lista
* Marcar tarefa como concluída (clicando na tarefa)
* Salvar todas as tarefas no navegador (LocalStorage) para que permaneçam após fechar o site
* Carregar tarefas salvas ao abrir a página
* Visualização de progresso ou pontuação se quiser adicionar gamificação

Sugestão visual:

* Cada tarefa com borda colorida conforme prioridade
* Tarefas concluídas riscadas ou esmaecidas
* Layout responsivo para celular e desktop

Tecnologias:

- HTML
- CSS
- JavaScript
